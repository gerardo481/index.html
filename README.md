# Image Toolkit

A free, browser-based image utility tool. No installs, no sign-ups — just open and use.

## Features

- **Format Conversion** — Convert images to JPEG, PNG, or WebP with adjustable quality
- **Bulk Upload** — Drag & drop or browse multiple images at once
- **Sequential Renaming** — Auto-rename files with a custom prefix and number sequence (e.g. `GO123-001.jpg`, `GO123-002.jpg`)
- **Background Color Fill** — Apply a white or black background before converting
- **eBay HTML Links** — Generate `<img>` HTML tags from a base URL for eBay listings
- **Imgur Upload** — Upload images directly to Imgur and get shareable `https://` URLs
- **CSV Export** — Export filenames and Imgur URLs in a single-row CSV, with all URLs pipe-separated, ready for spreadsheet use

## How to Use

1. Open the app in any modern browser
2. Set your preferred output format, quality, rename prefix, and background color
3. Drag & drop images or click to browse
4. Choose an action:
   - **Convert & Process** — converts and downloads files locally
   - **Upload to Imgur & Get URLs** — uploads to Imgur and returns shareable links
5. Export results as CSV or copy URLs to clipboard

## Supported Input Formats

JPG, PNG, WebP, GIF, BMP, TIFF

## Output Formats

JPEG, PNG, WebP

## Notes

- All processing happens in your browser — no images are sent to any server except when using the Imgur upload feature
- Imgur uploads are anonymous and public
- No account or API key required
